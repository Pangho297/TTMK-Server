경매를 통해 원하는 가격으로 중고물품을 거래하세요!

# 💾 Tech Stack

## Common
- ESlint
- Dotenv
- Axios
- Socket.io

## Front
- TypeScript
- SASS(SCSS)
- Styled-Component
- React
- React Hooks
- React Route Dom
- React Responsive
- Redux
- Typesafe-Action

## Back
- Node.js
- Express
- Multer
- MySQL
- Sequelize
- AWS(EC2, ELB, RDS, S3, Route53, CloudFront)
- Docker
