우리동네 경매장 땅땅마켓에서

- SASS
- Styled-Component
- React
- React Hooks
- React Route Dom
- React Responsive
- Redux
- Typesafe-Action

- AWS(EC2, RDS, S3, Route53, CloudFront)
- Docker
